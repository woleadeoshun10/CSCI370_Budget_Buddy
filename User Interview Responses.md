# Budget Buddy — User Interview Summary

## Overview
This document summarizes insights from 14 user interviews. The goal was to understand spending habits, budgeting behaviors, pain points, and which motivational features resonate across different financial literacy levels.

---

# User Responses

## Respondent 1
- Salaried worker, limited budgeting experience  
- Wants simple savings tools + reminders  
- Rated all motivational features **5/5**  
- Impulsive: **1–10 regrettable purchases/month**

---

## Respondent 2
- Hourly worker, strong financial knowledge  
- Prefers simple spending limits  
- Wants clearer spending breakdowns  
- Motivation features rated **4/5**  
- Not impulsive

---

## Respondent 3
- Student earning ~$25/hr  
- Basic financial knowledge  
- Uses Budget Buddy already  
- Impulsive (1–10 regrets)  
- Responds well to nudges + reminders

---

## Respondent 4
- Student budgeting weekly on paper/iPad  
- Wants overdraft warnings + future goal alignment  
- Regrets 1–10 purchases/month  
- Work-hours: **5/5**, Goal delay: **3/5**

# Key Themes

### 1. Impulse Control Matters
Most users have **1–10 regrettable purchases/month**.  
They want:
- friction before purchases  
- reminders  
- spending limits  
- work-hours translation  

### 2. Different Literacy Levels, Similar Needs
Even users with strong financial knowledge struggle with:
- impulsivity  
- consistency  
- staying aligned with long-term goals  

### 3. Most Motivating Features
Common high-rated motivators:
- **Achievements**  
- **Work-hours translation**  
- **Goal delay**  
- **Friendly nudges**  

Least motivating:
- **Streaks** (low across many users)
